  1. object allInOne extends Product with Serializable

    The input sequence data are two paired-end datasets.

    The input sequence data are two paired-end datasets. The standard notation is (R1, R2) for a dataset.

    Let's call the two datasets A and B, so that we have 4 read collections: (A.R1, A.R2), (B.R1, B.R2). For both A and B, R1 reads are both better (quality-wise) and larger than those in R2.

    Datasets are mixed in both R1 and R2 fastq files. We need to identify them based on their patterns (see below).
